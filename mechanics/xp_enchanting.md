# Experience & Enchanting

## XP Sources
- Killing mobs: 1-20 XP depending on mob type
- Mining ores: coal 0-2, lapis 2-5, redstone 1-5, diamond 3-7, emerald 3-7
- Smelting: 0.1-1.0 per item
- Breeding: 1-7 XP
- Fishing: 1-6 XP
- Trading with villagers: 3-11 XP depending on trade

## XP Levels
- XP orbs collected -> fill XP bar -> level up
- Level 0-16: 2*level + 7 XP per level
- Level 17-31: 5*level - 38 XP per level
- Level 32+: 9*level - 158 XP per level
- Total to reach level 30: 1395 XP

## Enchanting Table
- Place within ring of bookshelves (1 block gap, max 15 shelves = level 30 enchants)
- 3 options shown with level cost + one guaranteed enchantment visible
- Costs: slot 1 = 1 level + 1 lapis, slot 2 = 2 + 2, slot 3 = 3 + 3
- Enchantments are semi-random, seed changes when any enchant is applied

## Key Enchantments

### Weapons
- Sharpness I-V: +1/+1.5/+2/+2.5/+3 melee dmg
- Smite I-V: +2.5 per level vs undead
- Bane of Arthropods I-V: +2.5 per level vs spiders/silverfish
- Knockback I-II: extra knockback on hit
- Fire Aspect I-II: sets target on fire 4/8 sec
- Looting I-III: +1 max drop per level
- Sweeping Edge I-III: more sweep dmg

### Tools
- Efficiency I-V: faster mining (+2/+4/+6/+8/+10 speed)
- Unbreaking I-III: chance to not consume durability
- Fortune I-III: more ore/crop drops
- Silk Touch: mine block itself (grass, ores, glass, etc)
- Mending: XP repairs item instead of filling bar

### Armor
- Protection I-IV: -4% dmg per level (max 16% per piece)
- Fire Protection I-IV: -8% fire dmg per level
- Blast Protection I-IV: -8% explosion dmg per level
- Projectile Protection I-IV: -8% projectile dmg per level
- Feather Falling I-IV: -12% fall dmg per level (boots only)
- Respiration I-III: +15 sec underwater breath per level (helmet)
- Aqua Affinity: normal mining speed underwater (helmet)
- Depth Strider I-III: faster underwater walk (boots)
- Thorns I-III: reflect 1-4 dmg on attacker (costs durability)

### Bow
- Power I-V: +25% dmg per level (150% at V)
- Punch I-II: extra arrow knockback
- Flame: fire arrows, ignite target
- Infinity: no arrow consumption (need 1 arrow in inventory)

### Universal
- Unbreaking I-III: all tools/armor
- Mending: all tools/armor (mutually exclusive with Infinity on bow)

## Anvil Combining
- Combine enchanted items or apply enchanted books
- Cost in XP levels, increases with each operation
- Max cost cap: 39 levels (too expensive beyond that)
- Combine same enchant = +1 level (if same level)
