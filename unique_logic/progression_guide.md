# Player Progression Guide

## Phase 1: Survival (Ring 0, first 30 min)

### Immediate Goals
1. Punch tree, get wood, craft wooden tools
2. Find or dig stone, craft stone tools
3. Kill sheep for wool + bed (sleep = set spawn point)
4. Find coal, make torches
5. Build small shelter (4x4, roof, door, torches)
6. Farm wheat seeds from grass, start small wheat farm near water
7. Find exposed iron ore on cliff faces, mine with stone pick
8. Craft iron tools + bucket

### Key Milestones
- Bed placed and slept in (spawn set)
- Iron pickaxe crafted
- Basic food supply (bread or cooked meat)
- Water source secured (bucket + 2 water sources = infinite)
- Explored 2-3 nearby islands

## Phase 2: Expansion (Ring 0-1, hours 1-3)

### Goals
1. Build first bridge to neighboring island
2. Explore 5-10 islands, note biomes and resources
3. Find lapis lazuli (ring 1) for enchanting
4. Build enchanting setup: enchanting table + bookshelves
5. Enchant iron armor and tools
6. Build animal pen: bring 2 cows/pigs/sheep via bridge
7. Start breeding animals for reliable food

### Key Milestones
- First bridge completed
- Enchanting table operational
- Animal farm producing food
- Explored ring 0 and entered ring 1
- Full iron armor set

## Phase 3: Infrastructure (Ring 1-2, hours 3-8)

### Goals
1. Build bridges deeper into ring 1, then ring 2
2. Find and claim a large island as main base
3. Build proper base: storage room (chests), farm, animal pen, enchanting room
4. Find gold ore (ring 1+), start making golden apples
5. Find diamond ore (ring 2, deep in large islands)
6. Craft diamond pickaxe (minimum), save other diamonds
7. Create obsidian (water bucket + lava source)
8. Build enchanting table (4 obsidian + 2 diamond + 1 book)

### Key Milestones
- Main base established on good island
- Diamond pickaxe crafted
- Enchanting table with bookshelves (15 for max level)
- First enchanted diamond tool
- Bridge network spanning 3-5 islands

## Phase 4: Brewing (Ring 3+, hours 8-15)

### Goals
1. Reach ring 3, find volcanic island
2. Raid volcanic island for blaze rods (need at least 2: 1 for brewing stand, 1 for powder)
3. Find nether wart (volcanic island chest or soul sand patch)
4. Build brewing stand, start potion production
5. Critical potions: fire resistance (for volcanic raids), healing, strength
6. Find void crystals on island undersides (ring 3+)
7. Craft grappling hook for safer traversal
8. Build outpost chain: bed + chest every 3-4 islands along path

### Key Milestones
- Brewing stand operational
- Fire resistance potions available
- Nether wart farm growing
- Grappling hook crafted
- Outpost chain to ring 3+

## Phase 5: Deep Exploration (Ring 5-10, hours 15-30)

### Goals
1. Full diamond armor, enchanted (Protection IV, Unbreaking III)
2. Diamond sword with Sharpness V, Looting III
3. Prepare expedition loadout for high-ring runs
4. Build reinforced/enclosed bridges in ring 5+ (wind protection)
5. Craft wind charm for traversal
6. Find and raid ore-rich islands
7. Build ender chest for safe resource banking
8. Establish forward base in ring 5+

### Expedition Loadout
- Diamond armor (enchanted)
- Diamond sword + bow (enchanted)
- Stack of golden apples
- Fire resistance + healing + strength potions
- Grappling hook
- 3-4 stacks of cobblestone (bridge building)
- Bed (set spawn before dangerous island)
- Ender chest (bank valuable finds immediately)
- Food (steak stack)
- Torches
- Bucket of water (fall damage prevention, fire extinguishing)

### Key Milestones
- Full enchanted diamond gear
- Ender chest acquired
- Ring 10 reached
- Void crystals farmed, void resistance potions available
- Multiple outpost bases across rings

## Phase 6: Endgame (Ring 10+, hours 30+)

### Goals
1. Push deeper into rings for increasingly rare resources
2. Build obsidian bridges for permanent infrastructure
3. Craft island anchors for outpost protection
4. Challenge extreme-level mobs for bragging rights
5. Collect precious blocks from high-ring islands
6. Build mega-base showcasing collected resources
7. Help other players (multiplayer): build community bridges, trade resources

### Endgame Activities
- Ring 50+ expeditions: diamond block harvesting
- Speed runs: how fast can you reach ring 20 from fresh spawn
- Building projects: massive bridges, sky cities, floating structures
- Community: maintaining bridge highways for newer players
- Mob farming: optimized spawner-based XP/loot farms
- Village optimization: maximizing villager trades
- Exploration: mapping entire ring 0-5, cataloging all islands

## Death Recovery Strategy

### Low Ring Death (0-5)
- Walk back along bridges
- Items probably recoverable (5 min timer)
- Mild setback

### Mid Ring Death (5-15)
- Run back via bridges (5+ min travel time)
- Items may despawn before arrival
- Bank valuables in ender chest before risky activities
- Set bed at forward outpost

### High Ring Death (15+)
- Potentially catastrophic if bed was at high ring and got destroyed
- Items almost certainly lost (too far to reach in 5 min)
- Strategy: never carry irreplaceable items to high rings
- Emergency recall item: craft and carry for emergencies
- Accept losses, rebuild from outpost supplies

### Void Death
- Items in void = gone forever
- Most punishing death type
- Prevention: sneak on bridges, avoid island edges, respect wind
- Void resistance potion: buys time but doesn't save items
